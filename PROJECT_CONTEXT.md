## Git: дві гілки з різним оформленням

Обидві гілки мають однаковий контент сторінки (тексти, адреси з Instagram, галерея, карти). Відрізняється лише **фон**:

| Гілка | Опис |
|--------|------|
| **`master`** | Класичний суцільний темний фон без фото на всю сторінку. Зараз це гілка, з якої зазвичай **збирається GitHub Pages** (`push` у `master` → workflow). |
| **`experiment/background`** | Додано атмосферний фон **`photo/back.png`**: шари `.page-bg` / `.page-bg__image` / `.page-bg__scrim` у `index.html` та `styles.css`, напівпрозорі секції з `backdrop-filter`. |

**Відкат «тихішого» варіанту фону** (якщо знадобиться): git-тег **`background-mix-subtle`** вказує на коміт до підсилення видимості фону; у `styles.css` після `.page-bg__scrim` є коментар **«Резерв фону»** зі старими значеннями `filter` і градієнта.

Щоб залишити на сайті лише один варіант: злити обрану гілку в `master` (або перемкнути деплой — залежить від політики репозиторію).

## Технічне

- Карти: вбудовані **Google Maps** (iframe) у картках напрямів + посилання «Відкрити в Google Картах».
- Репозиторій: **git@github.com:Hruzduk/Yorik.git**; основні гілки: **`master`**, **`experiment/background`**.
- Деплой: **GitHub Pages** через workflow `.github/workflows/pages.yml` (у налаштуваннях репо Pages → джерело **GitHub Actions**).

При змінах зберігати стиль: темна тема, червоний акцент, українська мова контенту.
